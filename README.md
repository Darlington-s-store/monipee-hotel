# Monipee Hotel

A modern, luxury hotel management and booking system for Monipee Hotel in Ahansowodee, Ghana.

## Features

- **Public Website**: Beautiful, responsive hotel website with rooms, amenities, gallery, and contact pages.
- **Booking System**: Streamlined room booking process for guests.
- **Customer Dashboard**: Guest portal for managing reservations and profile.
- **Admin Dashboard**: Comprehensive management interface for rooms, bookings, customers, reviews, and content.
- **Authentication**: Secure login/registration for guests and administrators.

## Tech Stack

- **Frontend**: React, TypeScript, Tailwind CSS
- **Framework**: Vite
- **UI Components**: Shadcn UI, Radix UI, Lucide Icons
- **State Management**: React Query, Context API

## Getting Started

1.  Clone the repository
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```

## Project Structure

- `/src/pages`: Application pages (Public, Auth, Admin, Dashboard)
- `/src/components`: Reusable UI components
- `/src/contexts`: Global state providers (AuthContext)
- `/src/lib`: Utilities and helper functions

## Admin Access

To access the admin panel, navigate to `/admin/login`.
